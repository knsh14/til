## 本日のハイライト

研究と実装の双方で「エージェント基盤」と「自動運転プラットフォーム」が同時に動いており、両領域で実用化に向けた設計議論が進展している。

1. **[Claude agent — Phase 11: customizations / plugins](https://github.com/microsoft/vscode/pull/318113)** (microsoft/vscode) — VS Code 内蔵 Agent Host の大型フェーズで、customizations / plugins の拡張機構を追加。エージェント基盤の OSS 実装が成熟期に入っている代表例。
2. **[QUEST：完全合成タスクによるフロンティア・ディープリサーチエージェントの訓練](https://arxiv.org/abs/2605.24218)** (arxiv cs) — 8K の合成タスクでオープン重みエージェント (2B〜35B) をプロプライエタリ水準まで引き上げ、訓練レシピと生成パイプラインを完全公開。Phase 11 のような OSS エージェント実装に学習レシピが追いついてきた。
3. **[モデルは自身の憲法をどの程度遵守するか？](https://arxiv.org/abs/2605.24229)** (arxiv cs) — Claude Sonnet 4.6 と GPT-5.2 の仕様書遵守率を 205/197 命題に分解して定量評価し、敵対的監査でも違反率が前世代の 1/3〜1/5 に低下することを確認。Agent Host が前提とする「振る舞いの予測可能性」を支える研究。
4. **[SEIDM：自動運転挙動のための安全かつ効率的なインテリジェントドライバモデル](https://arxiv.org/abs/2605.23915)** (arxiv eess) — 適応的安全係数で減速項を動的調整する IDM 拡張。安全性を犠牲にせず交通流効率を改善。
5. **[Renault: Initial platform support and Renault 5 E-Tech port](https://github.com/commaai/opendbc/pull/3361)** (commaai/opendbc) — Renault プラットフォーム初期サポートおよび Renault 5 E-Tech 個別車種対応。論文側 (SEIDM) の制御モデル研究と並行して、実車対応も着実に拡大。
6. **[`uv` tried to use wheels which don't exist](https://github.com/astral-sh/uv/issues/16841)** (astral-sh/uv) — uv キャッシュの整合性破壊問題。ext4 上の fsync/rename 挙動を巡る根本原因解析が継続中で、Python ツールチェイン基盤の信頼性議論として注目。
7. **[\[WIP\] XAP — eXtensible Application Protocol](https://github.com/qmk/qmk_firmware/pull/13733)** (qmk/qmk_firmware) — QMK ホストツールとファームウェアを接続する次世代プロトコル。長期 WIP だが進展ごとに周辺ツールへの波及が大きい。

## arxiv 新着論文

> 詳細: [arxiv/2026-05-27.md](../arxiv/2026-05-27.md)

カテゴリごとの注目論文を抜粋する。

### Computer Science (cs)
- **[QUEST：完全合成タスクによるフロンティア・ディープリサーチエージェントの訓練](https://arxiv.org/abs/2605.24218)** — 8K 合成タスクでオープン重みエージェント (2B〜35B) を訓練し、プロプライエタリなフロンティアシステムに匹敵する性能を達成。
- **[モデルは自身の憲法をどの程度遵守するか？](https://arxiv.org/abs/2605.24229)** — 公開仕様書を 205/197 の原子命題に分解した敵対的監査で、Claude Sonnet 4.6 と GPT-5.2 の違反率が前世代の 1/5〜1/3 に低下したことを定量化。

### cs.AI
- **[LLM 推論におけるどれだけの思考が必要か：冗長性の定量化と理解](https://arxiv.org/abs/2605.23926)** — 最先端推論モデルで思考トレースの 61〜93% が削っても正答に至れることを示し、過剰思考が報酬設計の構造的帰結であると理論証明した。

### cs.RO
- **[Afford-VLA：内在化されたアフォーダンスによる行動整合型ビジュアルプランニング](https://arxiv.org/abs/2605.24203)** — VLA 内部の学習可能トークンでタスク条件付きアフォーダンスマップを生成し、LIBERO で SOTA を達成、実機実験でも強い結果を確認した。

### Electrical Engineering and Systems Science (eess)
- **[SEIDM：自動運転挙動のための安全かつ効率的なインテリジェントドライバモデル](https://arxiv.org/abs/2605.23915)** — 適応的安全係数で減速項を動的調整する IDM 拡張。ACC やシミュレーション交通モデルへの直接実装が可能。

### Statistics (stat)
- **[因果性は人工知能の統計的良心：Pearl の梯子から信頼できる機械へ](https://arxiv.org/abs/2605.24076)** — 機械学習における因果推論の位置付けを Pearl の枠組みで整理し、信頼できる AI 構築の指針を提示。

### Quantitative Biology (q-bio)
- **[知識ガイド表現分解によるタンパク質構造-機能関係の学習](https://arxiv.org/abs/2605.23960)** — タンパク質の構造-機能関係をドメイン知識でガイドする表現分解手法を提案。
